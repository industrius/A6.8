# A6.8

Практическое задание

Напишите html код с прогрессбаром, тремя кнопками и пометками: “+1%”, “+3%” и “+7%”. Нажатие на соответствующую кнопку увеличивает значение прогрессбара. Простейший вариант решения: сделать три функции, каждая из которых увеличивает общую переменную со значением прогресса на соответствующее значение.

В поле ответа вставьте ссылку на Github Pages со свёрстанным решением.
